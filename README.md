# k40 GRBL cntrlr
 K40 ESP32 GRBL Controller.
 
This is a replacement for the M2 Nano controller. 
It supports GRBL 1.1 and raster engraving with grayscales.
Since it board is based on ESP32 it supports Serial, Bluetooth and Wifi communications as well as working offline through the use of an SDCard. 

Firmware is forked from https://github.com/bdring/Grbl_Esp32
